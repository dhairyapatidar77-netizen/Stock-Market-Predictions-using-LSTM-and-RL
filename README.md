# Stock Market Prediction Using LSTM and RL

This project explores stock market prediction using **Long Short-Term Memory (LSTM)** neural networks and reinforcement learning concepts.  
The main goal is to study historical stock data, prepare it for sequential learning, and build a model that can predict future stock price movement.

---

## Project Objective

The objective of this project is to:
- Understand stock market time-series behavior.
- Prepare historical data for deep learning.
- Build an LSTM model for prediction.
- Explore reinforcement learning concepts for decision support.
- Document the development process week by week.
